My Service

Web Design

Web Design adalah stilah umum yang digunakan untuk mencakup bagaimana isi web konten ditampilkan, yang dikirimkan ke pengguna akhir melalui World Wide Web, dengan menggunakan sebuah browser web atau perangkat lunak.

Web Developer

Web developer adalah peran yang memungkinkan untuk membuat sebuah situs. Mereka tahu bagaimana cara membuat situs dari nol, menulis kode apabila dibutuhkan, dan memastikan apa yang diinginkan pengguna.

UI/UX Designer

UI Designer fokus ke tampilan aplikasi, sedangkan UX Designer fokus ke perasaan atau pengalaman pengguna ketika menggunakan aplikasi. Selain itu, perbedaan keduanya ada pada ranah tugas yang dikerjakan tentunya.
